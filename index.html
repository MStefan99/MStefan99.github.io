Projects
Deep learning robot
Inspired by previous experiment, a deep learning snake AI, this project is demonstrating the deep neural network in action. The robot is placed inside a simple maze and with the help of only the neural network running on the server (no path finding algorithms involved) it is able to find the way through the maze towards the finish line.
The project consists of two parts: a robot firmware written in C (GitHub) and the server software made using Python and Tensorflow library (GitHub) connected together via Wi-Fi over MQTT protocol.
