Projects

  • Deep learning robot

    Inspired by the previous experiment, a deep learning snake AI, this project is demonstrating the deep neural network in action. The robot is placed inside a simple maze and with the help of only the neural network running on the server (no path finding algorithms involved) it is able to find the way through the maze towards the finish line.

    The project consists of two parts: a robot firmware written in C and the the server software made using Python and Tensorflow library connected together via Wi-Fi over MQTT protocol.

  • Midi Parser

    Developed for future projects, Midi Parser is a small C++ library for parsing binary MIDI files (SMF) to be later used in any C++ software.

  • Blog App

    A web app allowing to browse a small blog, featuring "favorite" list for each user complete with user registration, email verification, password reset, user settings, post search by tags or keywords, dynamic theme and REST API.

    Project also includes a separate Android app made with Kotlin allowing you to browse the blog right from your mobile device.

  • roomBot

    A small Discord bot which allows you to create and manage private channels (or "rooms") on public Discord servers, making it possible to chat or talk to other users privately right on the server, without needing to add them as friends.
